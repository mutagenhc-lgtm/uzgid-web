# Как выложить UzGid в интернет (бесплатно)

Цель: получить ссылку вида `https://uzgid.vercel.app`, которую можно
давать туристам и зашивать в QR-код. Бесплатно, навсегда. ~20 минут.

В папке три файла — не меняйте их структуру:
```
uzgid-web/
├── index.html        ← само приложение
├── api/
│   └── chat.js       ← прослойка, прячет ваш ключ
└── DEPLOY.md         ← этот файл
```

## Почему нельзя просто «закинуть файл»

Если вписать ключ Anthropic в сайт, его украдут и потратят ваши деньги.
Файл `api/chat.js` решает это: он живёт на сервере Vercel, держит ключ в
секрете, а сайт обращается к нему. Вам не нужно ничего в нём понимать —
просто не удаляйте папку `api`.

## Шаги

### 1. Заведите аккаунт Vercel
- Откройте https://vercel.com → «Sign Up».
- Проще всего войти через аккаунт GitHub (если его нет — Vercel предложит
  создать, это бесплатно и быстро).

### 2. Загрузите проект
Самый простой способ для новичка — через GitHub:
- Создайте на https://github.com новый репозиторий (кнопка «New»).
- Нажмите «uploading an existing file» и перетащите туда `index.html` и
  папку `api` целиком.
- Commit (зелёная кнопка внизу).

### 3. Подключите Vercel к репозиторию
- В Vercel: «Add New…» → «Project» → выберите ваш репозиторий → «Import».
- Никаких настроек менять не нужно, Vercel сам всё поймёт.

### 4. ГЛАВНЫЙ ШАГ — добавьте секретный ключ
Перед нажатием «Deploy» (или сразу после, в Settings проекта):
- Откройте раздел **Environment Variables**.
- Имя (Name): `ANTHROPIC_API_KEY`
- Значение (Value): вставьте ваш ключ Anthropic (тот самый, из бота).
- Сохраните.

### 5. Deploy
- Нажмите «Deploy». Подождите минуту.
- Vercel выдаст ссылку вида `https://uzgid-xxx.vercel.app` — откройте,
  проверьте: выберите язык, задайте вопрос. Гид отвечает = всё работает!

## Если добавили ключ после первого деплоя
Зайдите в проект → «Deployments» → у последнего нажмите «…» → «Redeploy».
Это нужно, чтобы новый ключ «подхватился».

## Сделать красивую ссылку
В Settings → Domains можно поменять адрес на `uzgid.vercel.app` (если
свободно) или подключить свой домен вроде `uzgid.uz` (домен покупается
отдельно, ~10–15$/год — это уже по желанию).

## Создать QR-код для гостиниц
Когда ссылка готова — вбейте её на любом бесплатном генераторе QR
(например, qr-code-generator.com), распечатайте и договаривайтесь с
гостиницами и кафе. Турист сканирует → сразу попадает в гид.

## Расходы
- Хостинг Vercel: бесплатно (для вашего объёма с запасом).
- Платите только за ответы модели Anthropic — как и в боте, по факту
  использования. Приглядывайте за балансом на console.anthropic.com.
